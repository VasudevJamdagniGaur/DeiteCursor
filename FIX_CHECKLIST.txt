THE PROBLEM:
------------
❌ Clicking "Continue with Google" does nothing on mobile app
❌ No Google account picker appears
❌ No Chrome opens

THE CAUSE:
----------
Missing file: google-services.json

THE FIX:
--------
☐ Step 1: Open Firebase Console
   https://console.firebase.google.com/

☐ Step 2: Select your project
   Project: deitedatabase

☐ Step 3: Go to Project Settings
   Click gear icon (⚙️) → Project settings

☐ Step 4: Find your Android app
   Look for app with package: com.deite.app
   (If not found, add new Android app with this package name)

☐ Step 5: Download google-services.json
   Click "Download google-services.json" button

☐ Step 6: Place the file
   Location: android\app\google-services.json
   Full path: C:\Users\coolb\Cursor Projects\android\app\google-services.json

☐ Step 7: Verify (run in PowerShell)
   .\verify-firebase-setup.ps1

☐ Step 8: Rebuild the app
   npm run build
   npx cap sync android
   cd android
   .\gradlew assembleDebug

☐ Step 9: Install new APK
   Find APK in: android\app\build\outputs\apk\debug\app-debug.apk
   Install on your phone

☐ Step 10: Test
   Open app → Click "Continue with Google" → Should work!

WHAT WILL WORK AFTER:
----------------------
✅ Native Google account picker will appear
✅ Select account → Instant sign-in
✅ Works like Reddit, Instagram, etc.
✅ No need for Chrome or browser
✅ Seamless native experience

VERIFICATION:
-------------
After placing google-services.json, run:
   .\verify-firebase-setup.ps1

This will check:
  - File exists in correct location
  - Package name is correct (com.deite.app)

  - Firebase dependencies are installed
